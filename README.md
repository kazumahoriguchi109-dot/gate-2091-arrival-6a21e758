# 7 Rooms / 7 Doors

PC向けの7部屋構成・謎解きサイトです。

## 構成

- `index.html`: 画面の骨組み
- `styles.css`: 雰囲気づけ、部屋ごとのビジュアル、レスポンシブ対応
- `script.js`: 進行、正誤判定、バッドエンド、探索メモ、セーブ、簡易BGM

## 7部屋の流れ

1. `白い封鎖室`
   A-Dの順番で読む導入問題
2. `反転記録室`
   鏡文字を読む問題
3. `停止時計室`
   順番と鏡読みを組み合わせて年号を出す問題
4. `無人食堂`
   頭文字を拾って状況を読む問題
5. `空庭温室`
   指定文字目を抜き出す問題
6. `避難記録庫`
   年代順に並べて意味を読む問題
7. `帰還ゲート室`
   これまでのルールを組み合わせる最終問題

## 機能

- 各部屋に1つずつバッドエンド
- 進行に応じて増える探索メモ
- ローカルストレージによる進行保存
- 部屋ごとに変化する簡易BGM
- 1ファイル完結の静的サイト構成

## 遊び方

`index.html` をブラウザで開くだけで動きます。

## 公開方法

### GitHub Pages

1. このフォルダを GitHub リポジトリに push
2. リポジトリの `Settings > Pages` を開く
3. `Deploy from a branch` を選ぶ
4. `main` ブランチの `/ (root)` を指定して保存
5. 数分待つと静的サイトとして公開される

`.nojekyll` を入れてあるので、そのまま GitHub Pages に載せられます。

### Netlify / Vercel

- このフォルダをそのままドラッグ&ドロップするだけで公開可能
- ビルド設定は不要
